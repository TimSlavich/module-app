Виконав: студент групи ІР-31 Славич Тимофій

Завдання

Напишіть програму для розрахунку витрат для некромантів на виклик та утримання створінь. Вартість виклику та утримання буде залежати від типу створіння та його характеристик.

Умови: Програма має підтримувати різні типи створінь, такі як зомбі, скелети, вампіри, демони тощо. Кожне створіння має власні характеристики, такі як сила, витривалість, рівень загрози, магічні властивості тощо.
Витрати для некромантів розраховуються відповідно до таких умов:
Для зомбі: (Ставка за виклик * Сила + Витривалість) * Коефіцієнт зомбі
Для скелетів: (Ставка за виклик * Сила + 2 * Витривалість) * Коефіцієнт скелета
Для вампірів: (Ставка за виклик * Сила + Витривалість + Рівень загрози) * Коефіцієнт вампіра
Для демонів: (Ставка за виклик * (Сила + Витривалість) * (1 + Магічні властивості)) * Коефіцієнт демона
